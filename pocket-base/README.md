# PocketBase Personalizado

Este directorio contiene los archivos personalizados de tu instancia de PocketBase.

## Cómo usar

1. Asegúrate de tener Node.js instalado en tu sistema
2. Abre una terminal en esta carpeta
3. Ejecuta: `node install.js`

El script se encargará de:
- Descargar el archivo ZIP personalizado de PocketBase
- Extraer los archivos necesarios
- Preparar la instancia de PocketBase

## Acceso al panel de administración

Una vez extraídos los archivos, puedes iniciar PocketBase con el comando apropiado para tu sistema.
Por defecto, el panel de administración estará disponible en:
https://zeus-basedatos.fly.dev/
